# ECR Plasma 기반 반도체 공정 기술 발표 웹페이지

어두운 배경의 공학 발표 스타일로 구성한 단일 파일(`index.html`) 웹 프로젝트입니다.  
주제는 **ECR(Electron Cyclotron Resonance) Plasma 기반 반도체 공정 기술**이며, 발표에 필요한 핵심 섹션을 포함합니다.

## 포함된 섹션

- 원리 (ECR 공명 및 플라즈마 형성 메커니즘)
- 장점
- 단점과 한계
- 공정 응용
- 인터랙티브 시뮬레이션 (자기장/파워/압력 슬라이더)
- 추후 동향

## 실행 방법

### 방법 1) 파일 직접 열기
1. 저장소 루트에서 `index.html` 파일을 더블 클릭하거나 브라우저에 드래그합니다.

### 방법 2) 로컬 서버 실행 (권장)
아래 명령으로 간단한 정적 서버를 실행할 수 있습니다.

```bash
python3 -m http.server 8000
```

이후 브라우저에서 아래 주소로 접속하세요.

- http://localhost:8000

## 프로젝트 구조

```text
.
├── index.html   # HTML/CSS/JavaScript가 모두 포함된 발표 페이지
└── README.md    # 프로젝트 설명 및 실행 방법
```

## 특징

- 단일 파일 배포: 설치 의존성 없이 즉시 실행 가능
- 공학 발표 스타일 UI: 다크 테마, 카드 레이아웃, 고정 네비게이션
- 실시간 인터랙션: 공정 파라미터 변화에 따른 지표 시각화(canvas 기반)
