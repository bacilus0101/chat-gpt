# HTML Topic Page Prompt Template

Use this prompt to generate another topic page for the presentation.

```text
Create a single static HTML file for the topic: <TOPIC_NAME>.

Requirements:
1. Use semantic HTML5 (`doctype`, `head`, `body`, headings, paragraph, and bullet list).
2. Match this style baseline:
   - Simple inline CSS
   - `Arial, sans-serif`
   - Content width around 780px and centered
   - A "back" link to `../index.html`
3. Include:
   - Title (`<h1>`)
   - 1 concise summary paragraph
   - 3 bullet points: concept, mechanism, application
4. Keep the language technical but concise.
5. Output only valid HTML.
```
